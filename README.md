# Toy store

Toy store is a Retro Toy  built with **React** and **Firestore** 
It loads menu items from firestore and allows itesm to be added and remove from cart, aswell as the removal, edit and add  new items to the store visa the admin view.

##  Live Demo

Try it here:  
**https://akevonwolfe.github.io/toy-store*

-React
-React-router-dom
-JavaScript
-Joi
-Zustand
-Firebase
-REST API integration
